# Cars List Project

## Requirements

- Build a boilerplate based on Webpack
- Compile a dummy local data
- Build a simple UI based on React Hooks with try/catch blocks

## Dependencies

- react
- react-dom
- prop-types
- bootstrap
- react-bootstrap

## Dependencies for Development Environment

- webpack
- webpack-cli
- webpack-dev-server
- html-webpack-plugin
- @babel/core
- @babel/preset-env
- @babel/preset-react
- babel-loader
- css-loader
- style-loader
- eslint-loader
- eslint
- eslint-plugin-react
